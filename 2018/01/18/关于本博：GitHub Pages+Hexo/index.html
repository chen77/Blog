什么是GitHub Pages&Hexo？

GitHub社区设计了Pages功能允许用户自定义项目首页，用以替代源码列表。GitHub Pages就是是用户编写的托管在GitHub上的静态网页。

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
